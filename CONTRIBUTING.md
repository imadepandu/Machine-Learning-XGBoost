## Cara Kontribusi

### 1. Fork Repository
- Klik tombol "Fork" di halaman repository
- Clone repository yang sudah di-fork ke komputer Anda

### 2. Buat Branch
```bash
git checkout -b feature/nama-fitur-anda
```

### 3. Lakukan Perubahan
- Buat perubahan yang diperlukan
- Pastikan kode mengikuti style yang konsisten
- Tambahkan komentar jika diperlukan

### 4. Commit Perubahan
```bash
git add .
git commit -m "Deskripsi perubahan yang jelas"
```

### 5. Push ke GitHub
```bash
git push origin feature/nama-fitur-anda
```

### 6. Buat Pull Request
- Buka halaman repository di GitHub
- Klik "New Pull Request"
- Jelaskan perubahan yang Anda buat

## Panduan Kode

### Style Python
- Gunakan PEP 8 style guide
- Maksimal 79 karakter per baris
- Gunakan 4 spasi untuk indentasi
- Tambahkan docstring untuk fungsi

### Naming Convention
- Variabel: `snake_case`
- Fungsi: `snake_case`
- Konstanta: `UPPER_CASE`
- Class: `PascalCase`

### Komentar
- Gunakan komentar yang jelas dan deskriptif
- Komentar dalam bahasa Indonesia atau Inggris
- Hindari komentar yang tidak perlu
